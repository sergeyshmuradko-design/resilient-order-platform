# 02 Application & JVM Details

Dashboard для детальной диагностики Spring Boot/JVM.

Порядок анализа:

1. HTTP latency и errors.
2. Tomcat busy threads.
3. Hikari active/pending connections.
4. Process CPU.
5. Heap и memory pools.
6. Allocation, promotion и GC pauses.
7. Live, daemon и peak threads.
